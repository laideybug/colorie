## Components List
* Arduino UNO R3
* ESP8266 ESP-01 WiFi module
* BH1750 light sensor
* AM2302 DHT22 temperature-humidity sensor
* DS1307 RTC module
* Soil moisture sensor
* Water level depth sensor
* KY-019 5V relay modules
